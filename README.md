# Rock Paper Scissors Game

An interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features

* Play against the computer
* Random computer choice generation
* Real-time score tracking
* Dynamic game result messages
* Interactive user interface
* Responsive layout

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Play

1. Click Rock, Paper, or Scissors.
2. The computer randomly selects a choice.
3. The winner is determined according to game rules.
4. Scores update automatically.

## Concepts Practiced

* DOM Manipulation
* Event Handling
* Functions
* Conditional Logic
* Arrays
* Random Number Generation
